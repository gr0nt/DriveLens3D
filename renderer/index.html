🔍 DriveLens

Version 0.1.0

A 3D disk usage visualization tool for Windows.
Explore your drives with six powerful visualization modes,
real-time scanning, S.M.A.R.T. diagnostics, and file management.

Built With

Electron  ·  Three.js r153  ·  WebGL  ·  PowerShell

Visualizations

3D Treemap  ·  Sunburst  ·  Bar Chart  ·  Stacked  ·  City  ·  Galaxy

Themes

Cosmos  ·  Neon City  ·  Ember  ·  Nature  ·  Ocean  ·  Aurora

Open source — MIT License

Tips

Hover any item to see its size and type in the right panel.
Right-click drives in All Drives view to run S.M.A.R.T. diagnostics.
Use the copy icon next to the breadcrumb to copy a path.
